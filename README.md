# 📚 Bookstore Manager CLI

Um sistema de linha de comando (CLI) para gerenciar uma livraria, desenvolvido em **TypeScript** com conexão ao **PostgreSQL**.

---

## 🚀 Funcionalidades
- Conexão com banco de dados PostgreSQL
- Estrutura organizada em `controllers`, `services`, `repositories`
- CLI interativo para gerenciar livros, clientes e empréstimos (em desenvolvimento)

---

## 🛠️ Tecnologias
- Node.js v18+
- TypeScript
- ts-node / ts-node-dev
- PostgreSQL
- dotenv
